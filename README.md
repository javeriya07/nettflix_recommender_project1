# 🎬 Netflix Movie Recommendation System

## 📌 Project Description
This project is a **Netflix Movie Recommendation System** built using Django and Machine Learning.  
It suggests movies to users based on similarity between movies using content-based filtering.

---

## 🚀 Features
✔ Recommend similar movies  
✔ User-friendly web interface  
✔ Fast and accurate suggestions  
✔ Built with Machine Learning model  

---

## 🛠️ Tech Stack
- Python
- Django
- Pandas
- NumPy
- Scikit-learn
- HTML, CSS
